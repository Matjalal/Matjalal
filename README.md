## 기술 스택 ⚡
- REST API
- SPRING BOOT (back-end)
- NEXT.JS (front-end)
- TypeScript
- Tailwind(style)
- JWT Token Auth
- DB(SQL)
- H2 DATABASE ( test DB )
- MariaDB ( main DB )
- @tanstack/react-query
- Swagger

## 기획 의도
- 서브웨이나 공차같은 커스텀 푸드 브랜드 내의 주문 조합에 대한 정보들이 한곳에 집중되어 있지 않고 분산되어 원하는 정보를 찾는데 불편함이 있습니다. 이러한 정보들을 담고 찾을 수 있는 서비스를 만들게 되었습니다.
